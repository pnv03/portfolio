# Pranav Hule - Portfolio Website

A modern, interactive portfolio website built with **React** and **Tailwind CSS** showcasing my professional experience, projects, skills, and ways to connect. The site emphasizes dynamic UI elements, personalized identity, and a unique presentation style.

---

## 🌟 Features

1. **Dynamic Hero Section**
   - Name switches between English and Devanagari.
   - Full-stack developer tagline with tech stack.
   - Terminal-style profile card with hover effects.

2. **Projects / Experience**
   - Showcases software development experience, internships, and hackathon projects.
   - Cards with animated hover effects and tags for tech stack.

3. **Skills & Tech Stack**
   - Clearly separated into Backend, Frontend, and DevOps.
   - Interactive skill cards with hover color effects.

4. **Connect Section**
   - Buttons for GitHub, LinkedIn, LeetCode.
   - Display-only Email and Phone for clarity.
   - Modern, responsive layout.

5. **Modern UI Design**
   - Subtle background gradients and blur layers.
   - Smooth scrolling and transition animations.
   - Typography and spacing optimized for readability.

6. **Responsive & Accessible**
   - Fully responsive design for mobile, tablet, and desktop.
   - Clear focus states and accessible color contrast.

---

## 🛠 Tech Stack

- **Frontend:** React, Tailwind CSS, Framer Motion (animations)
- **Languages:** JavaScript, TypeScript
- **Tools:** Git, GitHub, VS Code
- **Design:** Tailwind CSS, CSS animations, responsive layouts
